# Tools

General development utilities (fd, rg, jq, yq, shell, imagemagick, mermaid, dependency managers)

## Features

### Skills

| Skill | Description |
| --- | --- |
| `deps-install` | Universal dependency installer - auto-detect package manager |
| `fd-file-finding` | Fast file finding using `fd` |
| `rg-code-search` | Fast code search using `ripgrep` |
| `jq-json-processing` | JSON querying and transformation with `jq` |
| `yq-yaml-processing` | YAML querying and transformation with `yq` |
| `shell-expert` | Shell scripting and bash patterns |
| `imagemagick-conversion` | Image conversion and manipulation |
| `mermaid-diagrams` | Generate diagrams from text using Mermaid CLI (flowcharts, sequence, ERD, class) |

## Installation

From marketplace:

```text
/plugin install
tools@meaganewaller-marketplace
```

## Usage

Skills are invoked when relevant. Ask to **find files**, **search code**, **process JSON/yaml**, **generate diagrams**, **install dependencies**, **add a package**, or any variation therein.
